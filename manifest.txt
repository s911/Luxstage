Luxstage B2B Production Server Dependency Manifest
=================================================

Base OS
- CentOS 7/8 compatible Linux distribution (recommended: Rocky Linux 8 or AlmaLinux 8)

Web Tier
- Nginx 1.24+
- PHP 8.2+ (php-fpm)

PHP Extensions
- mysqli
- pdo_mysql
- mbstring
- xml
- gd
- zip
- opcache
- curl
- intl
- exif
- fileinfo
- sodium
- imagick (recommended)
- webp support via GD and/or Imagick

Database
- MySQL 8.0.x

Caching and Performance
- Redis 6/7 (object cache)
- CDN integration (Cloudflare recommended)

Security and Operations
- OpenSSL / TLS certificates (Let's Encrypt or commercial CA)
- Fail2ban (recommended)
- FirewallD or iptables
- Cron service enabled
- Log rotation (logrotate)

Mail Delivery
- SMTP relay (SendGrid, SES, or equivalent)

Build/Deploy Tooling
- Docker Engine 24+
- Docker Compose plugin 2.20+
- WP-CLI 2.10+
